# Solo Operator OS Commands

Status: active draft
Purpose: Commands and workflows for Solo Operator OS users.

## Start Command

```text
AXIS: SOLO START
```

Use this when setting up or starting the Solo Operator OS.

On first setup, the OS should ask whether the user wants to give the assistant/system a name.

## What Solo Operator OS Helps With

Solo Operator OS helps a solo operator manage:

- daily priorities
- business admin
- personal admin
- client work
- ideas
- decisions
- relationship memory
- follow-up
- weekly reviews
- 30-day improvement reviews

## Core Commands

| Command | What It Does |
|---|---|
| `AXIS: DAILY COMMAND` | Sets daily priorities and next actions |
| `AXIS: WEEKLY REVIEW` | Reviews the week and sets the next week |
| `AXIS: CAPTURE THIS` | Captures a note, idea, relationship, or action |
| `AXIS: DECISION` | Logs a decision and reason |
| `AXIS: 30 DAY REVIEW` | Reviews whether the OS is improving work and life |

## Daily Command Flow

Use:

```text
AXIS: DAILY COMMAND
```

The OS should ask or confirm:

1. What matters today
2. What is urgent
3. What can wait
4. What needs follow-up
5. What one action would move things forward

## Capture Flow

Use:

```text
AXIS: CAPTURE THIS
```

Then add the note.

The OS should classify it as one of:

- task
- idea
- relationship note
- decision
- admin item
- knowledge note
- future review item

## Decision Flow

Use:

```text
AXIS: DECISION
```

The OS should capture:

- decision
- reason
- options considered
- risk
- next action
- review date if needed

## Weekly Review Flow

Use:

```text
AXIS: WEEKLY REVIEW
```

The OS should review:

- what was completed
- what slipped
- what still matters
- what relationships need attention
- what admin needs clearing
- what should be the next priority

## Good Example Prompts

```text
I feel scattered. Help me choose today's top three priorities.
```

```text
Capture this idea and turn it into a next action.
```

```text
Review my week and tell me what I should focus on next.
```

```text
I need to decide whether to launch this offer now or test it first.
```

## Boundaries

Solo Operator OS can support planning, admin, drafting, and decision clarity.

It does not replace:

- legal advice
- financial advice
- tax advice
- medical advice
- therapy
- safeguarding support
- regulated professional judgement

## Best Use

Use it daily for small decisions and weekly for direction.

The value comes from returning to the system, not from one perfect setup session.
