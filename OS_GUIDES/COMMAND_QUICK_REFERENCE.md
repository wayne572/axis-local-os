# Command Quick Reference

Status: active
Purpose: Short command list for Axis OS v3 Enhanced.

## Universal Commands

| Command | Use |
|---|---|
| `Status` | Show current status |
| `Workstreams` | Show live workstreams |
| `Next action` | Get one next step |
| `Block list` | Show what is stuck |
| `Summarise this session` | Create a session summary |

## Mode Commands

| Command | Use |
|---|---|
| `Activate Orchestrate mode` | Plan, prioritise, decide |
| `Activate Build mode` | Create assets or systems |
| `Activate Lead mode` | Work on leads, outreach, pipeline |
| `Activate Learn mode` | Research and brief |
| `Activate Review mode` | Audit, refine, sign off |

## Setup Commands

| Command | Use |
|---|---|
| `AXIS: NEW CLIENT` | Start a Client OS setup |
| `AXIS: SOLO START` | Start a Solo Operator OS setup |

## Solo Operator Commands

| Command | Use |
|---|---|
| `AXIS: DAILY COMMAND` | Daily priorities |
| `AXIS: WEEKLY REVIEW` | Weekly review |
| `AXIS: CAPTURE THIS` | Capture note, idea, task, or relationship |
| `AXIS: DECISION` | Log a decision |
| `AXIS: 30 DAY REVIEW` | Review whether the OS is working |

## Copy Commands

| Command | Use |
|---|---|
| `AXIS: DRAFT LANDING PAGE` | Draft landing page |
| `AXIS: REVIEW COPY` | Improve copy |
| `AXIS: POSITIONING CHECK` | Check offer clarity |
| `AXIS: FINAL COPY PASS` | Final wording pass |

## Relationship Commands

| Command | Use |
|---|---|
| `Run relationship pulse` | Show follow-ups and relationship opportunities |
| `Capture relationship note` | Record relationship context |
| `Track warm intro` | Track an introduction opportunity |
| `Draft follow-up message` | Draft follow-up |
| `Review intro consent` | Check whether an intro is appropriate |

## Current System Commands

| Command | Use |
|---|---|
| `Update Technology Radar` | Check new AI, automation, model, compliance, and access-channel changes |
| `Run AI ROI review` | Check whether a workflow or client OS is proving value |
| `Review tool choice` | Use the Tool Stack Decision Matrix |
| `Run client governance check` | Check AI use, consent, human review, and data boundaries |
| `Create client-safe version` | Use the Client Version Generator Checklist |
| `Plan message capture` | Use the Voice And Message Capture Plan |

## Plain English Also Works

You do not need to remember every command.

You can say:

```text
Help me work out what to do next.
```

```text
Turn this into a workflow.
```

```text
Review this before I send it.
```

```text
Who should I follow up with?
```
