## Core Commands

| Command | What it does |
|---|---|
| `Status` | DCoS reads the tracker and reports current state |
| `Workstreams` | DCoS lists active workstreams and next actions |
| `Block list` | Lists anything stuck or waiting |
| `Next action` | Gives the single next concrete action |
| `Summarise this session` | Routes to session summary |
