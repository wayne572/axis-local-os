## Core Product

AXIS is an AI operating system, not a chatbot.

It gives Claude a structured role, operating rules, workflows, memory files, command triggers, review gates, and handover templates.

The current build is designed for:
- Wayne's internal SF&W Project Solutions operating system
- SME client operating systems after an audit
- Solo Operator OS for solo entrepreneurs and personal operating support
