# 02 - Capability Map

Status: active draft
Purpose: List what AXIS can currently do.

## 1. Operating Modes

AXIS works in five modes:

| Mode | Purpose | Output |
|---|---|---|
| ORCHESTRATE | Prioritise, route, plan, decide | next-action set |
| BUILD | Create assets, documents, workflows, templates | finished artefact |
| LEAD | Prospecting, outreach, pipeline movement | qualified pipeline movement |
| LEARN | Research and market intelligence | one-page brief |
| REVIEW | Audit, validate, refine, sign off | decision or remediation list |

## 2. DCoS Work Control

AXIS uses Delivery Control System discipline.

It tracks work through:
- active
- in review
- locked
- archived

This prevents work from becoming vague or orphaned.
