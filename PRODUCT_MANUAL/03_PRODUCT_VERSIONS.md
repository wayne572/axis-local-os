Rule:
These should not become separate AIs.

## Product Rule

Each version must preserve the same logic:
- clear role
- structured context
- command triggers
- workflow routing
- trackers
- review gates
- boundaries
- handover discipline

The product can be simplified for a user, but the operating logic should not be dumbed down.
