# Electrician Website Demo

Status: active demo
Purpose: Show how Axis turns a simple business need into a quote-ready website concept.

## User Input

```text
I am an electrician. I need a simple site so customers understand what I do and can request a quote.
```

## Axis Interpretation

The customer needs a simple service website, not a complicated brand campaign.

The website must make these things clear:

- what electrical services are offered
- who the service is for
- how customers request a quote
- what information is needed before pricing
- why the electrician can be trusted
- what happens after enquiry

## Website Structure

1. Hero with clear offer and quote CTA.
2. Services section.
3. Quote request form.
4. How it works.
5. Trust and safety section.
6. Contact CTA.

## Quote Flow

The quote form asks for:

- name
- service type
- job details
- location and deadline in the message
