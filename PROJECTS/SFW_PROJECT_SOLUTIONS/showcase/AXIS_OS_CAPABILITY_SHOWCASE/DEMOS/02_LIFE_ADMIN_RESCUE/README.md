# Demo 02 - Life Admin Rescue

Status: active draft

## Demo Input

```text
I have bills, appointments, forms, renewals, and family stuff everywhere. I do not know what matters first.
```

## Axis Output

### Sort

| Bucket | Meaning |
|---|---|
| Today | Deadline, money risk, appointment risk |
| This week | Needs movement soon |
| Parked | Useful but not urgent |

### First Actions

1. List anything with a deadline.
2. Pick the two items with consequences if ignored.
3. Do the smallest possible next action for each.

### Suggested Admin Tracker

| Item | Category | Deadline | Next Action | Risk If Ignored | State |
|---|---|---|---|---|---|
| Bill review | Bills | Confirm | Check due dates | Late fee / stress | active |
| Appointment booking | Health / family | Confirm | Book or reschedule | Missed care / delay | active |
| Renewal | Documents | Confirm | Find renewal date | Cover gap / cost | active |

## Demo Value

Shows usefulness outside business. This is for ordinary life, not just entrepreneurs.
