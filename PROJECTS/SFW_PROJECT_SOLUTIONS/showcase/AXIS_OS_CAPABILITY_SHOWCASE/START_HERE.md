# Start Here - Axis OS Capability Showcase

Status: active

## First Goal

Build a demonstration environment that makes people say:

```text
I can see exactly how this would help me.
```

## What To Open First

1. `AXIS_OS_DEMO_WEBSITE.html`
2. `DEMO_SCENARIOS.md`
3. `OUTCOME_EXAMPLES.md`
4. `YOUTUBE_VIDEO_PLAN.md`
5. `CLAUDE_SCRIPT_CREATION_BRIEF.md`

## Live Demo Command

Use this when recording or presenting:

```text
AXIS: SHOWCASE START
```

The showcase should then ask:

```text
Who are we demonstrating for: solo operator, life admin, small business, creator, career, SME, audit, or web build?
```

## Demo Flow

Use this sequence for most videos:

1. Show messy input.
2. Ask Axis to organise it.
3. Show the output.
4. Show the tracker, page, dashboard, or website that came from it.
5. Explain the value in one sentence.

## First Build Sprint

Complete these first:

- core demo folder structure
- outcome examples
- visual website showcase
- YouTube video plan
- one full web build example
- one full life admin example
- one full small business example
