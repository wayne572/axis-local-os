# Daily Outreach Runbook - Wayne Francis / SF&W Project Solutions

**Owner:** Wayne Francis
**Time required:** 30 minutes/day, Mon-Fri
**Goal:** 50 contacts/week -> 200/month -> 1+ deployment/month minimum

---

## The Rules (Non-Negotiable)

1. **Same time, same place every weekday.** Pick it now. Mine: ___________
2. **30 minutes max.** Timer on. When it rings, stop - sent or not.
3. **Sent beats polished.** No rewriting after the third pass.
4. **No new system tweaks during this 30 min.** Outreach only.
5. **Log every send** in the tracker (bottom of this file).

---

## The 30-Minute Sequence

### 0-5 min - Open & Warm Up
- Open this file
- Open Gmail (wayne@waynefrancis.co.uk)
- Open the lead list for today (`D:/Wayne Francis/Outreach/Leads/YYYY-MM-DD.md`)
- Read yesterday's tracker. Any replies? Flag them - reply BEFORE new outreach.

### 5-20 min - Personalise & Draft (10 messages)
- For each of the 10 leads:
  - Skim their LinkedIn or company site (60 seconds max)
  - Pick ONE specific hook (a recent post, hire, service, location, vertical pain)
  - Drop into the template below
  - Run through humanize criteria mentally (no jargon, no hype, plain English)
  - Append signature block

### 20-28 min - Send
- Paste each into Gmail or LinkedIn
- Send. Move to next.
- Do NOT re-read after sending.

### 28-30 min - Log
- Update the tracker at the bottom of this file
- Close laptop. Done.

## Reply Handling (When They Bite)

| Reply type | Your move |
|---|---|
| "Tell me more" | Send 3-bullet summary + Calendly link. Don't over-explain. |
| "Not now" | "Totally fair. Mind if I check back in 90 days?" Log the date. |
| "Not interested" | "Appreciate the reply. Best of luck." Remove from list. |
| Question about price | "Pricing depends on whether you need an audit, a build, or ongoing optimisation. The current offer stack is in review, so I would size the requirement first before quoting." |
| Silence after 5 days | One follow-up: "Bumping this up - worth a 15-min look?" Then drop. |
