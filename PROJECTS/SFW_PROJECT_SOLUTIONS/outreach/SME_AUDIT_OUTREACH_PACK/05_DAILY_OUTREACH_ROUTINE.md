# Daily Outreach Routine

Status: active draft
Owner / brand: Wayne Francis
Trading entity: SF&W Project Solutions
Purpose: Daily routine for generating SME audit conversations.

## Time Required

30 minutes per working day.

## Daily Target

10 quality touches per day.

This can include:
- first messages
- follow-ups
- warm intro asks
- replies
- reactivations

## Routine

### 0-5 Minutes - Review

Open:
- `04_LEAD_TRACKER_TEMPLATE.md`
- inbox
- LinkedIn
- relationship memory if relevant

Check:
- replies
- follow-ups due
- booked calls
- do-not-contact requests

## 5-20 Minutes - Send Or Follow Up

Pick one action:
- send 10 first messages
- send 5 first messages and 5 follow-ups
- process replies and book calls
- ask for warm intros

Use:
- `01_CORE_OUTREACH_MESSAGES.md`
- `02_SECTOR_VARIANTS.md`

## 20-27 Minutes - Log

Update:
- lead tracker
- next action
- follow-up date
- status

## 27-30 Minutes - Close

Decide tomorrow's first action.

Write it here:

```text
Tomorrow's first outreach action:
```

## Weekly Rule

Every Friday, review:
- messages sent
- replies
- calls booked
- sectors responding
- best hook
- next week's focus

## Quality Rule

Do not chase volume at the cost of relevance.

One clear, relevant message is better than ten vague ones.
