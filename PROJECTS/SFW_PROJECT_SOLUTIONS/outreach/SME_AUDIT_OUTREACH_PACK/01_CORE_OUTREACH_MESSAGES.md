# Core Outreach Messages

Status: active draft
Owner / brand: Wayne Francis
Trading entity: SF&W Project Solutions
Purpose: Reusable outreach messages for the SME AI Automation Audit.

## Positioning

Lead with:
- admin time
- slow follow-up
- manual copy-and-paste
- scattered tools
- inconsistent client updates
- missed opportunities

Do not lead with:
- AI transformation
- disruption
- replacing staff
- generic productivity claims
- hype
