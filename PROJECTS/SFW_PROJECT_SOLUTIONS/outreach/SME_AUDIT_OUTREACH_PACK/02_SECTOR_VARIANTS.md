# Sector Variants

Status: active draft
Owner / brand: Wayne Francis
Trading entity: SF&W Project Solutions
Purpose: Sector-specific hooks for the SME AI Automation Audit.

## Rule

The offer stays SME-wide.

Only the examples and pain points change by sector.

## Recruitment Agencies

Likely pain:
- candidate follow-up
- client updates
- CV formatting
- interview scheduling
- vacancy intake
- CRM hygiene

Message hook:

```text
I help recruitment teams spot where candidate follow-up, client updates, and admin handoffs are slowing things down.
```

Audit angle:
Review candidate journey from first contact to interview follow-up.

First build ideas:
- candidate follow-up tracker
- vacancy intake assistant
- interview scheduling workflow
- client update draft system
