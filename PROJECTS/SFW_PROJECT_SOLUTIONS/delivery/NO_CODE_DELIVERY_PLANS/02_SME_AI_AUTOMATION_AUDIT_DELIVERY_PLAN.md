- poor handover

## Delivery Timeline

Recommended:
5-10 working days.

## Stage 1 - Fit Check

Goal:
Confirm there is a real workflow problem and the client is willing to share process details.

Ask:
- What workflow is causing the most admin?
- What gets copied, chased, or repeated?
- Where do follow-ups get missed?
- What tools are currently used?
- What would save the most time in the next 30 days?

If the client cannot name any workflow pain, do not sell the full audit yet.

## Stage 2 - Book Audit And Request Examples

Ask for:
- example enquiry
- example follow-up
- example report/quote/proposal/document
- current workflow steps
- tool list
- pain points from team

No-code method:
Client sends examples by email or shared folder.

## Stage 3 - Discovery Call

Use:

`audit/DISCOVERY_CALL_NOTES_TEMPLATE.md`

Capture:
- business context
- workflow pain
- tools
- buying signals
- current workaround
- next step

## Stage 4 - Workflow Audit

Use:

`audit/SME_AI_AUDIT_CHECKLIST_TEMPLATE.md`

Review 2-3 workflows.

Score:
- time saved
- revenue impact
- ease to build
- risk level
- priority

No-code method:
Use Claude to convert notes into the checklist.

Prompt:

```text
Using these discovery notes and examples, complete the SME AI Automation Audit Checklist. Prioritise practical workflows that can be improved without custom code first.
```

## Stage 5 - Create Audit Report

Use:

`audit/AUDIT_REPORT_TEMPLATE.md`

Report should include:
- executive summary
- workflows reviewed
- key findings
- quick wins
- recommended automation map
- recommended first build
- next step proposal
- boundaries

No-code method:
Use Claude to draft the report from the completed checklist, then Wayne reviews it.

## Stage 6 - Review Call

Walk the client through:
- what was found
- what matters most
- what not to build yet
- recommended first build
- whether Client OS Build is a fit

Do not pressure sell.

The audit should earn trust by being clear.

## Stage 7 - Proposal Decision

If there is a good build opportunity, move to:

`03_CLIENT_OS_BUILD_HANDOVER_DELIVERY_PLAN.md`

If not, give the client quick wins and close the audit.
