# No-Code Delivery Plans

Status: active draft
Owner / brand: Wayne Francis
Trading entity: SF&W Project Solutions
Purpose: Step-by-step delivery plans for AXIS offers that do not require Wayne to write code.

## Core Principle

These workflows are designed for no-code delivery.

Wayne should be able to deliver using:
- Claude
- folders
- markdown documents
- email
- calls
- simple trackers
- spreadsheets if needed
- Notion / Google Docs / Microsoft 365 if preferred by the client
- Telegram or WhatsApp later as access channels

No programming is required for the first version.

## Delivery Rule

Start simple.

Do not automate a process until the manual workflow is clear.

The first version should prove:
- the problem is real
- the workflow is useful
- the user understands how to run it
- the handover is clear
- the next review is booked

## Tool Rule

If a client already uses a tool, use that tool where possible.

Do not force a new platform unless there is a clear reason.
