# Active Loops

Status: active
Purpose: Prevent tasks, promises, decisions, and admin from disappearing.

## How To Use

Every open item should have:
- owner
- next action
- due date or review date
- state

Allowed states:
- active
- waiting
- in review
- locked
- archived
