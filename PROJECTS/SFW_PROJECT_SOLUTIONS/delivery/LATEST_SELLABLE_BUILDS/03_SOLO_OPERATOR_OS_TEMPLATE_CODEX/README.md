# Solo Operator OS Template - Codex

Status: production template - V4
Platform: Codex
Purpose: Personal / solo entrepreneur AI OS template.

## What This Is

This is a personal operating system for one person running inside Codex.

It helps them manage:
- work
- business
- personal admin
- relationships
- ideas
- weekly priorities
- follow-ups
- open loops

## What Makes It Different

This is not a general chatbot.

It has:

- command centre
- active trackers
- weekly review
- relationship memory
- personal admin workflows
- solo business workflows
- decision support
- V4 operating logic
- memory discipline
- governance/review rules
- care package operating standard
- AI OS control register
- freeze protocol
- source trust and prompt injection check
- monthly memory audit
- AI usage and ROI tracker
- platform change watch
- tone and style profile

Primary Codex instruction files:

- `AGENTS.md`
- `CODEX.md`

`CLAUDE_REFERENCE.md` is retained only as a cross-platform reference.
