# Client OS Growth Handover Script

Status: template
Suggested length: 60 to 90 minutes

## Opening

This system has been built around the workflows agreed after the Audit. It supports the team with visibility, follow-up discipline, delivery tracking, decision capture, and safe AI-assisted drafting.

It supports decisions. It does not replace the accountable people in the business.

## Walkthrough Order

1. Command Centre.
2. Pipeline and Follow-Up.
3. Client Memory.
4. Delivery Tracker.
5. Decision Log.
6. Templates and prompts.
7. Governance notes.
8. First 7 days adoption plan.
9. 30-day review.

## Role Practice

Owner:

- review command centre
- identify blockers
- log one decision
- run weekly review

Sales:

- add enquiry
- add follow-up
- use draft prompt

Operations / Delivery:

- update active work
- log blocker
- add client update note

Admin / Team:

- update assigned fields
- confirm data boundaries

## Close

Confirm:

- client can access the system
- users know where to start
- first 7 days plan is agreed
- 30-day review is booked
- change-control route is understood
