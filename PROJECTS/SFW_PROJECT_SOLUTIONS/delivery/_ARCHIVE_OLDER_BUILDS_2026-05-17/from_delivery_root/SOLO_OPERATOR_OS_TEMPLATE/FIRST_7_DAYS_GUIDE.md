# First 7 Days Guide

Status: template
Purpose: Help the solo client get value quickly without overfilling the system.

## Rule

Do not try to use every part of the OS at once.

Use it for real work, one issue at a time.

## Day 1 - Start

Type:

```text
AXIS: SOLO START
```

Complete the setup questions.

## Day 2 - Daily Priorities

Type:

```text
AXIS: DAILY COMMAND
```

Use it to choose today's top actions.

## Day 3 - Capture

Type:

```text
AXIS: CAPTURE THIS
```

Paste a messy note, idea, follow-up, or message.

## Day 4 - Admin

Type:

```text
AXIS: ADMIN CLEAR-DOWN
```

List admin that has been sitting too long.

## Day 5 - Business Or Project Action

Type:

```text
AXIS: BUSINESS BUILDER
```

Use it for one offer, content, sales, delivery, or follow-up action.

## Day 6 - Decision

Type:

```text
AXIS: DECISION
```

Use it for one decision you have been delaying.

## Day 7 - Weekly Review

Type:

```text
AXIS: WEEKLY REVIEW
```

Review what worked, what slipped, and what to focus on next.
