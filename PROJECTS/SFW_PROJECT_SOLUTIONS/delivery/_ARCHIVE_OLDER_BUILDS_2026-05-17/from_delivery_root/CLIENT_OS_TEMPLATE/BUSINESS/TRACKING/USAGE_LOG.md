# Usage Log

Status: optional active
Purpose: Track limited usage patterns to support the client and improve the system.

## Rule

Track only what is needed.

Do not paste full client content into this log unless the client has agreed and it is necessary for support.

## Tracker

| Date | User / Role | Module Used | Action Type | Outcome | Issue? | Improvement Note | Follow-Up |
|---|---|---|---|---|---|---|---|
| | | | | | | | |

## Action Type Options

Use:
- daily review
- weekly review
- tracker update
- email draft
- quote draft
- relationship note
- intro tracking
- workflow change
- support issue
- training question

## Outcome Options

Use:
- completed
- in review
- failed
- needs human review
- out of scope
- compliance hold

## Improvement Notes

Improvement notes should be short and non-sensitive.

Good:

```text
Client uses invoice chaser weekly. Add shortcut prompt.
```

Avoid:

```text
Full pasted client email with customer personal data.
```
