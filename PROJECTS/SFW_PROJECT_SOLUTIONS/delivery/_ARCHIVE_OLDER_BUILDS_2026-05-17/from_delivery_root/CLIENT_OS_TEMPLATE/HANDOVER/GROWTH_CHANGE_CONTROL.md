# Growth Change Control

Status: template
Use: Protect scope after the Audit and blueprint are approved.

## Included Without Change Request

- Minor wording changes.
- Clarifying fields already in approved modules.
- Small fixes found during testing.
- Corrections to sample records or training notes.

## Requires Change Request

- New workflow.
- New user group or permission model.
- New platform or integration.
- Migration from another system.
- Finance automation.
- Content engine.
- Custom reporting beyond approved command centre.
- Any regulated or high-risk automation.

## Rule

If the request changes the agreed workflow list, user roles, data boundaries, platform, or success measures, treat it as a change request.
