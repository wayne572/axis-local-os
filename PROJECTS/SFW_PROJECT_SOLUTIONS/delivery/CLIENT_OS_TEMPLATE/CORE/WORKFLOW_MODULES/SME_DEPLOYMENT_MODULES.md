# SME Deployment Modules

Status: active
Purpose: Generic workflow modules available for client delivery.

## Module 1 - Enquiry Follow-Up

Purpose:
Capture, acknowledge, and follow up new enquiries.

Outputs:
- acknowledgement message
- follow-up reminder
- lead status update
- owner assignment

## Module 2 - Quote Or Proposal Generator

Purpose:
Turn repeat quote or proposal work into a structured process.

Outputs:
- draft quote
- proposal outline
- follow-up email
- internal review checklist

## Module 3 - Standard Email Handler

Purpose:
Reduce time spent answering repeat questions.

Outputs:
- response bank
- draft replies
- escalation prompts
- communication checklist

## Module 4 - Invoice And Payment Chaser

Purpose:
Improve cash flow by making payment follow-up consistent and professional.

Outputs:
- reminder email
- overdue follow-up
- escalation notice
- payment tracker update

## Module 5 - Job, Client, Or Pipeline Tracker

Purpose:
Create a simple operating view of active work, enquiries, quotes, clients, and next actions.

Outputs:
- live tracker
- weekly status view
- overdue action list
- handoff notes

## Build Order

Recommended:

1. Tracker
2. Enquiry follow-up
3. Standard email handler
4. Quote or proposal generator
5. Invoice and payment chaser

## Scope Rule

Only activate modules included in the agreed client scope.
