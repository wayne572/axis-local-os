# User Guide

Status: active
Purpose: Simple guide for the client.

## What To Ask

Use this OS to:
- see what needs attention
- update workflows
- draft emails
- track follow-ups
- prepare weekly reviews
- keep decisions and actions visible

## Best Daily Prompt

```text
What needs my attention today?
```

## Best Weekly Prompt

```text
Run the weekly review.
```

## How To Add Notes

Paste notes in plain English.

Example:

```text
New enquiry from Sarah at ABC Ltd. Wants a quote by Friday. Follow up tomorrow.
```

Claude should update the right tracker or draft the next action.

## What Not To Do

Do not ask the OS to make legal, HR, medical, financial, or compliance decisions without human review.

Do not add sensitive personal data unless it is necessary for the agreed workflow.

## When To Ask SF&W

Ask SF&W if:
- a workflow is broken
- scope needs changing
- a new automation is needed
- compliance is unclear
- the client wants to add a new module
