# Simple User Guide

Status: template
Use: Give the solo client a plain-English starting guide.

## What This Is

This is a personal AI operating system for one person.

It helps organise priorities, admin, follow-ups, ideas, decisions, relationships, and weekly review.

It is not a normal chatbot. It is a working structure.

It works by turning messy input into:

```text
capture
-> plan
-> action
-> review
-> tracker or memory update
```

## How To Start

Type:

```text
AXIS: SOLO START
```

If this is the first session, complete the setup questions.

## Main Commands

```text
AXIS: DAILY COMMAND
```

Use this to decide what matters today.

```text
AXIS: CAPTURE THIS
```

Use this to turn a messy note, idea, message, or voice-note transcript into structure.

```text
AXIS: IDEA TO MEMORY
```

Use this when an idea might become a reusable plan, tracker item, or useful memory.

```text
AXIS: DECISION
```

Use this when thinking through a decision.

```text
AXIS: ADMIN CLEAR-DOWN
```

Use this for personal or business admin that needs sorting.

```text
AXIS: BUSINESS BUILDER
```

Use this for offers, leads, content, delivery, or business actions.

```text
AXIS: WEEKLY REVIEW
```

Use this once a week to close loops and plan the next week.

## Boundaries

This OS can help organise, draft, plan, review, and prioritise.

It does not replace professional legal, tax, financial, medical, therapeutic, safeguarding, employment, or regulated advice.

It should only remember useful, durable information. Temporary thoughts should stay as active loops, not permanent memory.
