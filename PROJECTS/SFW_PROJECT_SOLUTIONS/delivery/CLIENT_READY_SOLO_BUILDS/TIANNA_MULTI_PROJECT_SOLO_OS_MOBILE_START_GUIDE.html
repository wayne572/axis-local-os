Tianna Multi-Project Solo OS - Email Setup Guide

Hi Tianna,

This folder is your personal Axis AI operating system.

It is designed to help you manage multiple projects, ideas, follow-ups, decisions, and life admin without needing to keep everything in your head.

What You Have Received

You should receive a zip file called:


TIANNA_MULTI_PROJECT_SOLO_OS.zip

Inside it is a folder called:


TIANNA_MULTI_PROJECT_SOLO_OS

That folder contains your setup guide, daily command centre, project tracker, weekly review, decision log, and other support files.

Step 1 - Save The Zip File

Save the zip file somewhere easy to find, such as:

  • Downloads
  • Desktop
  • Documents
  • iCloud Drive
  • OneDrive
  • Google Drive

Choose a place you can access again easily.

Step 2 - Unzip The Folder

Right-click or tap the zip file and choose:


Extract All

or:


Unzip

After unzipping, you should see the folder:


TIANNA_MULTI_PROJECT_SOLO_OS

Step 3 - Keep The Folder Together

Do not move individual files out of the folder.

The system works best when everything stays together in one folder.

If you want to move it, move the whole folder.

Step 4 - Open This File First

Open:


CLIENT_STARTUP_GUIDE.md

This explains what the system does and how to start using it.

Step 5 - Complete The Setup Questions

Next, open:


SETUP_QUESTIONNAIRE.md

Answer the questions in simple notes.

You do not need perfect answers. The point is to get your projects and responsibilities out of your head and into the system.

Step 6 - Start With Your Projects

Open:


BUSINESS/SOLO_BUSINESS_TRACKER.md

Add your active projects.

For each project, write:

  • project name
  • status
  • priority
  • next action
  • deadline, if there is one
  • blocker, if there is one

Step 7 - Add Open Loops

Open:


TRACKERS/ACTIVE_LOOPS.md

Add anything unfinished, promised, pending, or mentally noisy.

Examples:

  • someone to message
  • something to book
  • a task you keep remembering
  • a decision you need to make
  • something you are waiting for

Step 8 - Use The Start Command

When using the folder with an AI assistant such as Claude or ChatGPT, use:


AXIS: TIANNA START

Then ask for the mode you need:

  • Daily focus
  • Project review
  • Weekly reset
  • Idea capture
  • Decision help
  • Admin clear-down

Best First Prompt

Use this:


AXIS: TIANNA START

Help me set up my active projects, open loops, life admin, and first weekly priorities.

If You Feel Overwhelmed

Use this:


I have too much going on. Review my projects, open loops, and admin, then help me choose one must-do, one should-do, and one small momentum action.

First Week Goal

For the first week, do not try to make the system perfect.

Just use it to capture:

  • active projects
  • open loops
  • follow-ups
  • admin
  • decisions
  • ideas

The first win is getting everything visible.

Important

This system is here to support your clarity and planning.

It does not make medical, legal, financial, employment, or safety decisions for you.

For important sensitive decisions, use it to organise your thoughts and then speak to the right person.

Tianna Setup Questionnaire

Status: setup required

Purpose: Gather the information needed to personalise Tianna's OS.

Current Projects

  • What projects are currently active?
  • Which projects matter most right now?
  • Which projects have deadlines?
  • Which projects are exciting?
  • Which projects feel heavy?
  • Which projects should probably be paused?

Responsibilities

  • What recurring responsibilities does Tianna manage?
  • What happens weekly?
  • What happens monthly?
  • What does she often forget or leave late?

Planning Style

  • Does Tianna prefer daily plans or weekly plans?
  • Does she like lists, tables, short prompts, or guided questions?
  • What makes planning feel overwhelming?
  • What makes her feel in control?

Follow-Ups And People

  • Who does Tianna need to keep in touch with?
  • What promises or follow-ups are currently open?
  • Are there any relationships linked to projects?

Life Admin

  • What personal admin is currently open?
  • Any renewals, appointments, forms, or bookings due?
  • What admin causes the most background stress?

Decision Support

  • What decisions are currently pending?
  • What choices keep coming back?
  • What does Tianna need help weighing up?

Tianna Solo OS Handover Notes

Status: ready for handover

Handover Goal

Tianna should leave the handover knowing:

  • where to start each day
  • how to add active projects
  • how to capture open loops
  • how to choose weekly priorities
  • how to park ideas without losing them
  • how to ask for decision support

Handover Agenda

  • Explain the purpose of the OS.
  • If sent by email, open EMAIL_SETUP_GUIDE.md.
  • Confirm the zip file has been unzipped and the folder is kept together.
  • Open START_HERE.md.
  • Run AXIS: TIANNA START.
  • Complete the first section of SETUP_QUESTIONNAIRE.md.
  • Add at least three active projects.
  • Add at least three open loops.
  • Add one life admin task.
  • Add one pending decision.
  • Run a mini weekly review.
  • Agree first week focus.

First Week Focus

Suggested first-week goal:

Get every active project, open loop, and admin task out of Tianna's head and into the system.

Success Signs

  • Tianna knows what to do first each day.
  • She can see her active projects.
  • She has fewer floating mental reminders.
  • She knows which projects are active and which are parked.
  • She can identify a next action without needing to rethink everything.
