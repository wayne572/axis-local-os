# Tianna Multi-Project Solo OS Modules

Status: client-ready draft
Purpose: Defines the working modules in Tianna's personal OS.

## Module 1 - Daily Command Centre

Purpose:

Help Tianna start the day with a clear view of priorities, project actions, follow-ups, and personal admin.

Outputs:

- top three actions
- must-do commitment
- project priority
- follow-up due
- admin task
- one small momentum action

## Module 2 - Multi-Project Tracker

Purpose:

Hold every active project in one place so Tianna can see status, next action, deadline, blocker, and priority.

Tracks:

- project name
- category
- status
- next action
- deadline
- blocker
- energy level
- priority

## Module 3 - Active Loops

Purpose:

Stop unfinished tasks and unresolved follow-ups from floating around in Tianna's head.

Tracks:

- open loop
- owner
- due date
- next action
- waiting on
- status

## Module 4 - Weekly Review

Purpose:

Reset the week, close loops, choose project priorities, and reduce clutter.

Outputs:

- wins
- unfinished loops
- projects to focus on
- decisions needed
- admin to clear
- people to contact
- parked ideas
- next week plan

## Module 5 - Idea To Action

Purpose:

Help Tianna capture ideas without immediately adding pressure.

Outputs:

- idea summary
- why it matters
- project it belongs to
- first small action
- park / test / build decision

## Module 6 - Decision Log

Purpose:

Support clearer choices when projects, timing, energy, and obligations compete.

Outputs:

- decision needed
- options
- trade-offs
- risks
- recommendation
- next action

## Module 7 - Life Admin Tracker

Purpose:

Keep personal admin visible and manageable.

Tracks:

- appointments
- forms
- bills
- renewals
- bookings
- household tasks
- documents

## Module 8 - Relationship Memory

Purpose:

Remember people, follow-ups, promises, and useful context.

Tracks:

- contact
- context
- last interaction
- follow-up date
- promise made
- useful notes

## Module 9 - Personal Knowledge Base

Purpose:

Store durable context Tianna should not need to repeat.

Stores:

- preferences
- recurring goals
- active project context
- personal rules
- important notes
