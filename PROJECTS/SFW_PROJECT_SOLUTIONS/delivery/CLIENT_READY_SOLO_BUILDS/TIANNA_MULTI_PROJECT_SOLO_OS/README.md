# Tianna Multi-Project Solo OS

Status: client-ready draft
Created for: Tianna
Purpose: Personal command system for a woman managing multiple active projects, responsibilities, ideas, and life admin.

## What This Is

This is Tianna's personal AI operating system.

It is designed to help her:

- see every active project in one place
- choose what matters this week
- avoid carrying everything in her head
- track open loops and follow-ups
- turn ideas into next actions
- manage life admin alongside project work
- make clearer decisions when too much is competing for attention

## Core Principle

Tianna should not have to remember everything.

The system should hold the moving parts so she can focus on the right next action.

## Best Use

Use this OS when Tianna needs to:

- plan the day
- review all current projects
- decide what to prioritise
- capture a new idea
- clear mental clutter
- prepare for the week
- track a follow-up
- make a decision
- organise admin


## First 7 Days

1. Open `EMAIL_SETUP_GUIDE.md` if this was received by email.
2. Open `CLIENT_STARTUP_GUIDE.md`.
3. Complete `SETUP_QUESTIONNAIRE.md`.
4. Add all active projects.
5. Pick the top three projects that matter most this week.
6. Add open loops and follow-ups.
7. Capture any unfinished ideas.
8. Run one weekly review.
9. Remove anything that does not need attention now.

## Boundary

This OS supports clarity, structure, and planning. It does not make personal, medical, legal, financial, or employment decisions for Tianna.
