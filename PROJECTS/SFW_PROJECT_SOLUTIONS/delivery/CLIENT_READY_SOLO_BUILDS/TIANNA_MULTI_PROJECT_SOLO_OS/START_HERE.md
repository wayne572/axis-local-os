# Start Here - Tianna Multi-Project Solo OS

Use this file when opening the system.

## Start Command

```text
AXIS: TIANNA START
```

## Choose A Mode

### Daily Focus

Use when Tianna needs to know what to do today.

Open:

- `COMMAND_CENTRE/DAILY_COMMAND_CENTRE.md`
- `TRACKERS/ACTIVE_LOOPS.md`
- `BUSINESS/SOLO_BUSINESS_TRACKER.md`

### Project Review

Use when Tianna needs to review multiple active projects.

Open:

- `BUSINESS/SOLO_BUSINESS_TRACKER.md`
- `DECISIONS/DECISION_LOG.md`
- `TRACKERS/ACTIVE_LOOPS.md`

### Weekly Reset

Use when Tianna needs to plan the week.

Open:

- `ROUTINES/WEEKLY_REVIEW.md`
- `COMMAND_CENTRE/DAILY_COMMAND_CENTRE.md`
- `TRACKERS/ACTIVE_LOOPS.md`

### Idea Capture

Use when Tianna has a new idea but does not know whether to act on it.

Open:

- `IDEAS/IDEA_TO_ACTION.md`
- `BUSINESS/SOLO_BUSINESS_TRACKER.md`

### Decision Help

Use when Tianna feels stuck between options.

Open:

- `DECISIONS/DECISION_LOG.md`
- `PERSONAL_CONTEXT.md`

### Admin Clear-Down

Use when life admin is creating background stress.

Open:

- `LIFE_ADMIN/PERSONAL_ADMIN_TRACKER.md`
- `TRACKERS/ACTIVE_LOOPS.md`

## First Setup Steps

1. Complete `SETUP_QUESTIONNAIRE.md`.
2. Add every active project to `BUSINESS/SOLO_BUSINESS_TRACKER.md`.
3. Add open loops to `TRACKERS/ACTIVE_LOOPS.md`.
4. Add recurring admin to `LIFE_ADMIN/PERSONAL_ADMIN_TRACKER.md`.
5. Run the weekly review once.
