# Tianna Multi-Project Solo OS - Client Startup Guide

## What This Helps With

This system helps Tianna manage multiple projects, ideas, admin tasks, follow-ups, and decisions without carrying everything in her head.

It is built to answer:

- What matters today?
- Which projects are active?
- What needs following up?
- What is still open?
- What can wait?
- What decision needs making?
- What should I focus on this week?

## Start Here

If you received this as a zip file by email, open this first:

`EMAIL_SETUP_GUIDE.md`

That guide explains how to save, unzip, and keep the folder together.

Use this command:

```text
AXIS: TIANNA START
```

Then choose one mode:

- Daily focus
- Project review
- Weekly reset
- Idea capture
- Decision help
- Admin clear-down

## Best First Step

Before using the system fully, complete:

`SETUP_QUESTIONNAIRE.md`

Then add:

1. Active projects into `BUSINESS/SOLO_BUSINESS_TRACKER.md`
2. Open loops into `TRACKERS/ACTIVE_LOOPS.md`
3. Admin into `LIFE_ADMIN/PERSONAL_ADMIN_TRACKER.md`
4. Pending decisions into `DECISIONS/DECISION_LOG.md`

## Daily Use

Open:

`COMMAND_CENTRE/DAILY_COMMAND_CENTRE.md`

Choose:

- one must-do
- one should-do
- one small momentum action

## Weekly Use

Open:

`ROUTINES/WEEKLY_REVIEW.md`

Review:

- what moved forward
- what stalled
- what needs a decision
- what can be paused
- what matters next week

## Important Rule

Not every idea needs to become a project.

Use:

`IDEAS/IDEA_TO_ACTION.md`

to decide whether to act now, test small, park, or delete.

## When Overwhelmed

Use this prompt:

```text
I have too much going on. Review my projects, open loops, and admin, then help me choose one must-do, one should-do, and one small momentum action.
```

## First 30 Days

For the first month, the aim is simple:

- get everything out of her head
- keep project status visible
- reduce missed follow-ups
- make weekly decisions easier
- stop inactive ideas creating pressure
