# Simple User Guide - Sonjie Solo OS

Status: active
Use: Plain-English starting guide.

## What This Is

This is a personal AI operating system for Sonjie.

It helps organise:

- rota and workplace admin ideas
- TikTok/content direction
- personal admin
- open loops
- decisions
- weekly review
- useful ideas

It is not a normal chatbot. It is a working structure.

## How To Start

Type:

```text
AXIS: SOLO START
```

If this is the first session, complete or review the setup questions.

## Main Commands

```text
AXIS: DAILY COMMAND
```

Use this to decide what matters today.

```text
AXIS: CAPTURE THIS
```

Use this to turn a messy note, idea, message, or voice-note transcript into structure.

```text
AXIS: IDEA TO MEMORY
```

Use this when an idea might become a reusable plan, tracker item, or useful memory.

```text
AXIS: CARE ROTA BUILDER
```

Use this for rota, shift, coverage, or workplace admin ideas.

```text
AXIS: TIKTOK REVIEW
```

Use this to check what your TikTok page is about, what to post next, and what needs to be clearer.

```text
AXIS: CREATOR PLAN
```

Use this to plan simple content themes and next posts.

```text
AXIS: DECISION
```

Use this when thinking through a decision.

```text
AXIS: ADMIN CLEAR-DOWN
```

Use this for personal or work admin that needs sorting.

```text
AXIS: WEEKLY REVIEW
```

Use this once a week to close loops and plan the next week.

## Boundaries

This OS can help organise, draft, plan, review, and prioritise.

It does not replace professional legal, tax, financial, medical, therapeutic, safeguarding, employment, HR, or regulated advice.

It also does not promise TikTok growth, income, virality, or guaranteed monetisation.
