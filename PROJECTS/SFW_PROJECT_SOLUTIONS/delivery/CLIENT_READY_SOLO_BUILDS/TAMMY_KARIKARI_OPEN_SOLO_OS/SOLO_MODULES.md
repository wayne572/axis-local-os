# Tammy Karikari Open Solo OS Modules

Status: open testing draft
Purpose: Defines the modules Tammy can test before the OS is locked.

## Module 1 - Daily Command Centre

Purpose:
Help the user start the day with focus.

Outputs:
- top three priorities
- urgent admin
- follow-ups due
- meetings or commitments
- one personal action
- one business action

## Module 2 - Weekly Review

Purpose:
Close loops and plan the next week.

Outputs:
- wins
- open loops
- decisions needed
- admin backlog
- relationship follow-ups
- next week focus

## Module 3 - Personal Admin Tracker

Purpose:
Stop life admin from disappearing.

Tracks:
- forms
- renewals
- bills
- bookings
- appointments
- documents
- household tasks

## Module 4 - Solo Business Tracker

Purpose:
Help a solo person move business work forward.

Tracks:
- offers
- leads
- content
- clients
- products
- launches
- revenue actions

## Module 5 - Relationship Memory

Purpose:
Remember people, follow-ups, intros, and useful context.

Tracks:
- contacts
- context
- follow-up dates
- intro opportunities
- important notes

## Module 6 - Idea To Action

Purpose:
Turn ideas into real next steps.

Outputs:
- idea summary
- category
- value
- next action
- minimum viable version
- due date

## Module 7 - Decision Support

Purpose:
Help the user make clearer decisions.

Outputs:
- options
- trade-offs
- risks
- recommended next step
- what to stop or delay

## Module 8 - Personal Knowledge Base

Purpose:
Keep durable context available.

Stores:
- preferences
- recurring goals
- personal rules
- project context
- important notes
