# Charity And CIC Handover Checklist

Status: active
Purpose: Confirms the charity/CIC AI OS is ready to hand over.

## Pre-Handover

- [ ] Organisation context completed
- [ ] Assistant name confirmed or default `Axis` accepted
- [ ] Agreed scope completed
- [ ] Included workflows listed
- [ ] Active workstreams populated
- [ ] Data handling notes reviewed
- [ ] Safeguarding boundary reviewed
- [ ] Usage tracking notice reviewed
- [ ] Client consent and analytics options recorded
- [ ] Email templates reviewed
- [ ] Prompt library reviewed
- [ ] User guide reviewed
- [ ] Out-of-scope items listed
- [ ] Organisation-specific facts checked

## Handover Session

- [ ] Explain what the OS does
- [ ] Explain what it does not do
- [ ] Show daily command centre prompt
- [ ] Show weekly review
- [ ] Show active workstream tracker
- [ ] Show charity/CIC module guide
- [ ] Show how to update a workflow
- [ ] Show how to draft and review messages
- [ ] Explain data and safeguarding guardrails
- [ ] Explain what requires human review
- [ ] Explain what usage is tracked and why
- [ ] Explain optional analytics and opt-out
- [ ] Explain when to contact SF&W

## Post-Handover

- [ ] First organisation test completed
- [ ] Organisation questions captured
- [ ] Any fixes logged
- [ ] First weekly review booked
- [ ] 30-day review booked
- [ ] Client OS marked `locked` or `in review`
