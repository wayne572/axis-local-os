# Charity And CIC Prompt Library

Status: active
Purpose: Useful prompts for the organisation to operate the OS.

## Wayne Introduction

```text
Create a warm introduction from Wayne Francis to [Person Name] at [Organisation Name].

Mention that Wayne currently runs a CIC himself, understands how hard it is to work in this sector, wants to give back where he can, and is offering this AI OS as a practical token of respect for the work they are doing.

Use the long introduction style only. Keep it personal, respectful, non-salesy, and reassuring.
```
