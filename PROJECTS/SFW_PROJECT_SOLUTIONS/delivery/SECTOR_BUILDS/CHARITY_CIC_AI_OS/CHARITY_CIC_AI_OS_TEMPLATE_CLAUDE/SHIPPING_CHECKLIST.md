# Charity And CIC AI OS Shipping Checklist

Status: template
Use: Complete before sending or handing over a charity/CIC client build.

## Package

- [ ] Organisation confirmed
- [ ] Legal structure recorded
- [ ] Named OS owner confirmed
- [ ] Instruction file updated for platform
- [ ] Client context completed
- [ ] Charity/CIC blueprint completed
- [ ] Included workflows added
- [ ] Excluded workflows documented
- [ ] Sample records added with fictional or anonymised data only
- [ ] Governance notes completed
- [ ] Safeguarding and data boundaries completed
- [ ] Templates/prompts reviewed
- [ ] Usage log ready
- [ ] First 7 days adoption path ready
- [ ] 30-day review path ready

## Testing

- [ ] Command centre tested
- [ ] New referral/enquiry tested
- [ ] Standard email drafting tested
- [ ] Funding/donor pipeline tested, if in scope
- [ ] Volunteer/partner memory tested, if in scope
- [ ] Service delivery tracker tested, if in scope
- [ ] Governance/board summary tested, if in scope
- [ ] Weekly review tested
- [ ] Sensitive-data boundary tested
- [ ] Out-of-scope request tested

## Handover

- [ ] Handover script prepared
- [ ] First 7 days adoption plan prepared
- [ ] Plain-English user guidance prepared
- [ ] Human review rules explained
- [ ] Organisation users confirmed
- [ ] First weekly review booked
- [ ] 30-day review booked
- [ ] Wayne reviewed final package
- [ ] Organisation owner reviewed final package
