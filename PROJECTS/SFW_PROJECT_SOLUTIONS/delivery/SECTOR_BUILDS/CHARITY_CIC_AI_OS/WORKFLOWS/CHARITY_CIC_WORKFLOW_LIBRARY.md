# Charity And CIC Workflow Library

Status: draft
Purpose: Reusable workflow modules for charity and CIC AI OS builds.

## 1. Daily Command Centre

Use for:
- today's priorities
- overdue follow-ups
- decisions needed
- blockers
- messages to draft
- meetings to prepare for

Example prompt:

```text
Run today's command centre from these notes and tell me what needs attention.
```

## Funding Integrity Rules

The OS must not:
- invent impact numbers
- fabricate beneficiary stories
- claim outcomes the organisation cannot evidence
- submit grant applications without human sign-off
- make legal, financial, eligibility, or safeguarding decisions
- pressure donors using manipulative language

The OS should always flag:
- missing evidence
- unsupported claims
- reporting deadlines
- funder restrictions
- match-funding requirements
- commitments that need trustee or senior approval
