## What This Is

This is Joseph's personal AI operating system.

It is designed to help Joseph manage the moving parts of Bluxe and his wider work without everything living in his head, inbox, WhatsApp, notes, or memory.

It is not a normal AI chat tool.

It is a structured working system for:

- daily priorities
- business tasks
- follow-ups
- ideas
- decisions
- relationship notes
- personal admin
- weekly review
- open loops

## What It Helps Joseph Do

This OS helps Joseph:

- decide what matters today
- turn messy thoughts into clear actions
- keep track of people to follow up with
- capture ideas before they disappear
- organise business admin
- make clearer decisions
- review the week properly
- spot what is being delayed or dropped
- keep Bluxe moving without needing everything to be perfect first

## How To Start

Open this folder in Claude.

This build is designed for Claude.

Then type:

```text
AXIS: SOLO START
```

The OS will start the setup flow.

On first setup, it should ask Joseph whether he wants to give the AI OS a name.

## Where The Client Guides Are

The client-facing PDF guides are in:

```text
CLIENT_DOCUMENTATION/
```

Use those PDFs for reading, review, and handover.

Keep the Markdown files in the main folder because Claude uses them as the working instruction layer.

## Main Commands

| Command | Use |
|---|---|
| `AXIS: SOLO START` | Start the OS |
| `AXIS: DAILY COMMAND` | Choose today's priorities |
| `AXIS: WEEKLY REVIEW` | Review the week and set direction |
| `AXIS: CAPTURE THIS` | Capture an idea, task, note, or follow-up |
| `AXIS: DECISION` | Think through a decision |
| `AXIS: ADMIN CLEAR-DOWN` | Clear down personal or business admin |
| `AXIS: BUSINESS BUILDER` | Work on offers, content, leads, sales, delivery, or growth |
| `AXIS: 30 DAY REVIEW` | Review whether the OS is helping |

## First Week

For the first week, Joseph should keep it simple.

Use the OS for:

1. Daily priorities
2. Capturing loose ideas
3. Logging follow-ups
4. Clearing admin
5. Making one decision at a time

Do not try to perfect the system on day one.

The value comes from using it regularly.

## Important Boundary

This OS helps with organisation, planning, drafting, decisions, and follow-through.

It does not replace professional advice for legal, tax, financial, medical, safeguarding, or regulated matters.

## Support

If the OS gives an answer that feels too broad, Joseph can say:

```text
Make this simpler and give me the next action only.
```

If it forgets the purpose of the system, Joseph can say:

```text
Read START_HERE.md and PERSONAL_CONTEXT.md again, then continue.
```
