# Start Here - Joseph / Bluxe Solo Operator OS

Status: Claude-ready client draft
Purpose: First page to open when Joseph starts his Solo Operator OS in Claude.

## Welcome

Joseph, this is your personal AI operating system.

It is here to help you run your work with more clarity, less scattered admin, and fewer dropped follow-ups.

You do not need to use technical language.

You can speak naturally, paste messy notes, or use the commands below.

This version has been prepared for Claude.

The PDF guides are stored in:

```text
CLIENT_DOCUMENTATION/
```

The Markdown files stay in the main folder so Claude can read the OS instructions properly.

## Start Command

Use:

```text
AXIS: SOLO START
```

The OS should then check:

`PERSONAL_CONTEXT.md`

If setup is incomplete, it should run first-time setup.

If setup is complete, it should ask:

```text
Do you want Daily Command Centre, Weekly Review, Capture, Decision Support, Admin Clear-Down, or Business Builder?
```

## Main Commands

```text
AXIS: DAILY COMMAND
```

Use this when you want to decide what matters today.

```text
AXIS: WEEKLY REVIEW
```

Use this to review the week, close loops, and choose the next week's focus.

```text
AXIS: CAPTURE THIS
```

Use this when you have a messy note, idea, voice note, message, or thought that needs turning into action.

```text
AXIS: DECISION
```

Use this when you need help thinking through options, risks, trade-offs, and next action.

```text
AXIS: ADMIN CLEAR-DOWN
```

Use this when admin is building up and you need to sort what matters first.

```text
AXIS: BUSINESS BUILDER
```

Use this for Bluxe work: offers, content, leads, sales, delivery, follow-ups, or growth actions.

```text
AXIS: 30 DAY REVIEW
```

Use this after a month to review whether the OS is helping.

## Operating Standard

Every session should create at least one of:

- a decision
- a next action
- an updated tracker
- a clearer plan
- a useful draft
- a closed loop

## Boundary

This OS supports organisation, planning, drafting, workflow management, and decision clarity.

It does not replace legal, financial, tax, medical, therapeutic, safeguarding, or regulated professional advice.
