# First 7 Days Guide

Purpose: Help Joseph get value from the OS quickly without overthinking the setup.

## Day 1 - Start

Type:

```text
AXIS: SOLO START
```

Answer the setup questions.

Choose whether to name the AI OS.

## Day 2 - Daily Priorities

Type:

```text
AXIS: DAILY COMMAND
```

Use it to choose today's top actions.

## Day 3 - Capture

Type:

```text
AXIS: CAPTURE THIS
```

Paste a messy note, idea, or follow-up.

Let the OS turn it into structure.

## Day 4 - Admin

Type:

```text
AXIS: ADMIN CLEAR-DOWN
```

List admin that has been sitting too long.

Ask the OS to sort it by urgency and effort.

## Day 5 - Business Builder

Type:

```text
AXIS: BUSINESS BUILDER
```

Use it for one Bluxe action: offer, content, sales, delivery, or follow-up.

## Day 6 - Decision

Type:

```text
AXIS: DECISION
```

Use it for one decision you have been delaying.

## Day 7 - Weekly Review

Type:

```text
AXIS: WEEKLY REVIEW
```

Review what worked, what slipped, and what to focus on next.

## Rule For The First Week

Do not try to fill every file.

Use the OS to solve real moments.

The system improves as you use it.
