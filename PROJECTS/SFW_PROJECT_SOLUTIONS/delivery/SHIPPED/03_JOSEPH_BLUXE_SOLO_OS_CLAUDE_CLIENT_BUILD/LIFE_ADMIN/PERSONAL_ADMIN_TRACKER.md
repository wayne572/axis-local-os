## Admin Clear-Down

Use this when the user feels overloaded:

1. List all admin items without judging them.
2. Mark anything urgent or time-sensitive.
3. Choose the smallest next action for each.
4. Pick three items to clear first.
5. Archive anything no longer relevant.

## Safety Note

For legal, tax, medical, debt, insurance, or regulated financial matters, help the user prepare questions and documents, but recommend qualified advice where needed.
