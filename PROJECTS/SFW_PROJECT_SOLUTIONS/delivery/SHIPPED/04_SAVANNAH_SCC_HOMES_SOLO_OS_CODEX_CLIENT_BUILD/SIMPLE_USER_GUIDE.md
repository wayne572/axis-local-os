# Savannah / SCC Homes - Simple User Guide

## What This Is

This is a personal AI operating system for Savannah and SCC Homes.

It helps organise work, follow-ups, guest messages, property tasks, ideas, decisions, and weekly priorities.

It is not a normal chatbot. It is a working system.

## How To Start

Open the folder in Codex and type:

```text
AXIS: SOLO START
```

If it is your first time, answer the setup questions.

## Main Commands

```text
AXIS: DAILY COMMAND
```

Use this at the start of the day. It helps decide what matters today.

```text
AXIS: GUEST TRIAGE
```

Use this for guest enquiries, replies, complaints, check-in questions, or review follow-ups.

```text
AXIS: PROPERTY OPS
```

Use this for cleaning, changeovers, maintenance, suppliers, property standards, and urgent property tasks.

```text
AXIS: CAPTURE THIS
```

Use this when you have a messy note, idea, task, or voice-note transcript.

```text
AXIS: DECISION
```

Use this when you need help thinking through a business or operational decision.

```text
AXIS: WEEKLY REVIEW
```

Use this once a week to close loops and plan the next week.

## Examples

You can say:

```text
AXIS: GUEST TRIAGE
Guest says the heating is not working and check-in was confusing. Help me reply and decide what to do next.
```

```text
AXIS: PROPERTY OPS
I need to chase the cleaner, confirm tomorrow's changeover, and get a maintenance quote for the bathroom tap.
```

```text
AXIS: CAPTURE THIS
I want to improve repeat bookings and maybe create a direct booking offer for business travellers.
```

## Important Boundaries

This OS can help organise, draft, plan, review, and prioritise.

It does not replace professional advice for legal, tax, insurance, health and safety, licensing, employment, or regulated matters.

If something feels legally or commercially sensitive, ask the OS to turn it into a review checklist and speak to the right professional.
