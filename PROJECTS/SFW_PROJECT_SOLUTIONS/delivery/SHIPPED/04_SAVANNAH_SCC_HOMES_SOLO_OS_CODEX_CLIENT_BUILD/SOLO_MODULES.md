# Savannah / SCC Homes Solo Operator OS Modules

Status: active
Purpose: Defines the modules for Savannah's Codex-based Solo Operator OS.

## Module 1 - Daily Command Centre

Purpose:
Help Savannah start the day with focus.

Outputs:
- top three priorities
- urgent guest or property items
- follow-ups due
- admin that must not slip
- one personal action
- one SCC Homes business action

## Module 2 - Weekly Review

Purpose:
Close loops and plan the next week.

Outputs:
- wins
- open loops
- guest issues
- property tasks
- decisions needed
- admin backlog
- relationship follow-ups
- next week focus

## Module 3 - Personal Admin Tracker

Purpose:
Stop life admin and business admin from disappearing.

Tracks:
- forms
- renewals
- bills
- bookings
- appointments
- documents
- household tasks
- business admin

## Module 4 - SCC Homes Business Tracker

Purpose:
Help Savannah move SCC Homes business work forward.

Tracks:
- bookings
- guest enquiries
- property tasks
- cleaning and changeovers
- maintenance
- suppliers
- reviews
- partnerships
- direct booking opportunities
- revenue actions

## Module 5 - Relationship Memory

Purpose:
Remember people, follow-ups, intros, and useful context.

Tracks:
- guests
- repeat guests
- cleaners
- maintenance contacts
- suppliers
- owners / landlords
- partners
- advisers
- follow-up dates
- important notes

## Module 6 - Idea To Action

Purpose:
Turn ideas into real next steps.

Outputs:
- idea summary
- category
- value
- next action
- minimum viable version
- due date

## Module 7 - Decision Support

Purpose:
Help Savannah make clearer decisions.

Outputs:
- options
- trade-offs
- risks
- recommended next step
- what to stop or delay

## Module 8 - Personal Knowledge Base

Purpose:
Keep durable context available.

Stores:
- preferences
- recurring goals
- personal rules
- project context
- important notes

## Module 9 - Service Accommodation Operations

Purpose:
Help Savannah keep guest, booking, property, cleaner, supplier, and maintenance work under control.

Tracks:
- guest enquiries
- check-in and check-out issues
- cleaning and changeover status
- maintenance requests
- property standards
- supplier follow-ups
- guest complaints
- review improvement actions
- compliance review items

Outputs:
- triaged issue list
- next action
- owner
- urgency
- follow-up date
- guest impact
- risk flag
