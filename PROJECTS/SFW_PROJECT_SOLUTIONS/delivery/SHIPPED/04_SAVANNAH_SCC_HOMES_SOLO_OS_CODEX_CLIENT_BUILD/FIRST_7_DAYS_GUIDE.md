# First 7 Days - Savannah / SCC Homes

Keep the first week simple.

The aim is to build the habit, not perfect the system.

## Day 1 - Start

Type:

```text
AXIS: SOLO START
```

Complete the setup questions.

## Day 2 - Daily Priorities

Type:

```text
AXIS: DAILY COMMAND
```

Use it to choose the top priorities for SCC Homes.

## Day 3 - Guest Messages

Type:

```text
AXIS: GUEST TRIAGE
```

Paste in one guest message, enquiry, complaint, or review issue.

Ask for:
- summary
- suggested reply
- next action
- follow-up date

## Day 4 - Property Tasks

Type:

```text
AXIS: PROPERTY OPS
```

List the property, cleaner, supplier, or maintenance items in your head.

Ask the OS to sort them by urgency.

## Day 5 - Capture Ideas

Type:

```text
AXIS: CAPTURE THIS
```

Capture a business idea, direct booking idea, guest experience improvement, or admin improvement.

## Day 6 - Decision

Type:

```text
AXIS: DECISION
```

Use it for one decision you have been delaying.

## Day 7 - Weekly Review

Type:

```text
AXIS: WEEKLY REVIEW
```

Review:
- what got done
- what slipped
- guest issues still open
- property tasks still open
- people to follow up with
- priorities for next week

## First Week Rule

Do not try to use every part of the OS at once.

Use it for real work, one issue at a time.
