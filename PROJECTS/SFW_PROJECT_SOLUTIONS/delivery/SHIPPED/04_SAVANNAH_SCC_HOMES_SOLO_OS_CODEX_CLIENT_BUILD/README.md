## What This Is

This is Savannah's personal AI operating system for SCC Homes.

It is designed to help Savannah manage the moving parts of a service accommodation business without everything living in her head, inbox, WhatsApp, notes, or memory.

It is not a normal AI chat tool.

It is a structured working system for:

- daily priorities
- guest enquiries and follow-ups
- bookings and admin
- property tasks
- cleaner, supplier, and maintenance coordination
- guest experience and review improvement
- ideas and decisions
- relationship notes
- weekly review
- open loops

## What Makes It Different

This is not a general chatbot.

It has:
- command centre
- active trackers
- weekly review
- relationship memory
- personal admin workflows
- solo business workflows
- decision support
- service accommodation operating workflows
