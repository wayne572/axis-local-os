# Daily Command Centre

Status: active
Purpose: Turn Savannah's day into a clear operating plan for SCC Homes and personal priorities.

## Trigger

```text
AXIS: DAILY COMMAND
```

## Inputs

Ask only what is needed:
- What is already booked today?
- Are there any check-ins, check-outs, guest messages, or property tasks today?
- What is urgent?
- What is weighing on your mind?
- What must not be dropped?
- How much energy do you have today: low, medium, or high?
