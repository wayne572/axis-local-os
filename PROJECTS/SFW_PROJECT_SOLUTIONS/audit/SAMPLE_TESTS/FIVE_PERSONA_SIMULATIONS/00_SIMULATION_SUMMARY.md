# Five Persona Simulation Summary

Status: in review
Purpose: Summarise five additional SME / solo operator funnel simulations.

## Simulations Run

| Simulation | Persona | Best Offer Route | Price Path | Fit |
|---|---|---|---|---|
| 1 | Solo consultant / freelancer | Solo Operator OS | GBP 3,995 | strong |
| 2 | Recruitment agency | Audit -> Client OS Build | GBP 5,595 + GBP 9,995 | strong |
| 3 | Property business | Audit -> Client OS Build -> Connector | GBP 5,595 + GBP 9,995 + GBP 3,195/qtr | strong |
| 4 | Accountancy firm | Audit -> Client OS Build | GBP 5,595 + GBP 9,995 | strong |
| 5 | Local services business | Audit -> Client OS Build | GBP 5,595 + GBP 9,995 | strong |

## What The Tests Proved

- The broad SME audit positioning works across multiple sectors.
- Sector examples should change, but the offer does not need to narrow.
- The strongest paid path is Audit -> Client OS Build.
- Solo Operator OS needs separate ownership-led positioning.
- Connector Mode is strongest as a post-build optimisation offer, not a cold standalone sale.

## Pricing Implications

Recommended pricing to use in proposal materials:

| Offer | Price | Positioning |
|---|---:|---|
| Solo Operator OS | GBP 3,995 | permanent operating system for solo operators |
| SME AI Automation Audit | GBP 5,595 | expert diagnosis and automation roadmap |
| Client OS Build And Handover | GBP 9,995 | custom implementation and handover |
| Relationship Connector Mode | GBP 3,195 per quarter | quarterly optimisation and growth support |

## Recommendation

Update proposal templates to include:
- build cost
- optional audit cost
- optional connector cost
- ROI logic
- ownership / no vendor lock-in statement
