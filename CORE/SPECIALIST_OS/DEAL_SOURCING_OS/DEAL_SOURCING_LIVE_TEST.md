# DEAL_SOURCING_LIVE_TEST.md

## 1. Status
Active - Live Test Phase

---

## 2. Purpose
This file defines a 7-day live execution test for the Deal Sourcing OS.

The goal is to:

- prove the system works
- generate real conversations
- build execution habit
- identify early improvements

---

## 3. Core Principle

This is not about perfection.

This is about:

- showing up daily
- following the system
- learning quickly

---

## 4. Test Duration

7 Days (no skipping)

Each day:

30-60 minutes max

---

## 5. Daily Targets

Every day:

- 5 new leads added
- 5 outreach messages sent
- 2 follow-ups sent (from previous days)

Minimum standard:

Do not go below this.

---

## 6. Day-by-Day Plan

### Day 1 - Setup + First Execution

- Define ICP (if not already done)
- Identify 5 leads
- Send 5 outreach messages
- Log everything

Goal:
Start momentum

---

### Day 2 - Repeat + Stabilise

- Add 5 new leads
- Send 5 outreach messages
- Send first follow-ups (if any)

Goal:
Build consistency

---

### Day 3 - Observe Responses

- Continue outreach (5 messages)
- Send follow-ups
- Track any replies

Goal:
See early signals

---

### Day 4 - Improve Messaging Slightly

- Continue outreach
- Adjust 1 small part of message (only ONE change)

Goal:
Start optimisation

---

### Day 5 - Maintain Flow

- Continue outreach
- Follow up consistently

Goal:
No drop in execution

---

### Day 6 - Qualification Focus

- Continue outreach
- Start identifying:
  - good leads
  - weak leads

Goal:
Improve targeting

---

### Day 7 - Review + Learn

- Complete outreach
- Review full week

Record:

- total messages sent
- responses received
- conversations started
- qualified leads

---

## 7. Tracking Requirements

Every action must be logged:

- lead added
- outreach sent
- follow-up sent
- response received
- outcome

No tracking = failed test

---

## 8. Success Criteria

Test is successful if:

- outreach was sent every day
- tracking was consistent
- at least 1-3 conversations started

---

## 9. Failure Conditions

Test fails if:

- days are skipped
- outreach stops
- tracking is ignored
- system is not followed

---

## 10. Expected Outcomes

You should expect:

- some messages ignored (normal)
- some replies (positive sign)
- at least a few conversations

---

## 11. Adjustment Rules

After Day 7:

- improve ICP slightly
- refine messaging slightly
- keep what worked

Do NOT:

- change everything
- rebuild the system

---

## 12. Discipline Rule

The system works only if:

you follow it fully for 7 days

No shortcuts.

---

## 13. End State

After 7 days:

- you understand the system
- you see real responses
- you have early opportunities
- you have data to improve

This is where deal flow begins.
