# MARKETING_POST_CREATION.md

## 1. Status
Active - Execution Sub-File

---

## 2. Purpose
Define how to create simple, effective posts.

---

## 3. Core Principle

Clarity over creativity.

---

## 4. Structure

1. Hook
2. Value
3. Simple close

---

## 5. Execution Steps

- start with a clear statement
- explain simply
- end with a thought or action

---

## 6. Example

Hook:
"Most people struggle to get clients because..."

Value:
Explain the problem simply

Close:
"Start with this instead."

---

## 7. Output Requirement

Posts must be:

- clear
- easy to read
- relevant

---

## 8. Failure Conditions

- too long
- unclear message
- no structure

---

## 9. End State

Posts are easy to create and understand.
