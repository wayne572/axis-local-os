# Brand Architecture

Status: active
Purpose: Defines how Wayne Francis and SF&W Project Solutions should be presented.

## Core Rule

Wayne Francis is the public brand.

SF&W Project Solutions is the trading entity.

Use this structure in all external-facing copy, offers, client documents, and product materials.

## Preferred External Presentation

Primary brand:
Wayne Francis

Trading entity:
SF&W Project Solutions

Recommended wording:

```text
Wayne Francis, trading under SF&W Project Solutions
```

Short version:

```text
Wayne Francis / SF&W Project Solutions
```

## What This Means

Use Wayne Francis when referring to:
- public profile
- personal brand
- expertise
- thought leadership
- client trust
- founder-led positioning
- speaking, content, and outreach

Use SF&W Project Solutions when referring to:
- trading entity
- contracts
- delivery
- invoices
- client agreements
- compliance documents
- formal business operations

## Product Relationship

AXIS is the operating system / product logic.

AXIS should not replace Wayne Francis as the brand.

AXIS can be described as:
- developed by Wayne Francis
- delivered through SF&W Project Solutions
- used to build practical AI operating systems for SMEs and solo operators

## Avoid

Do not present:
- Axis AI as the trading brand
- AXIS as a standalone company unless Wayne creates one later
- SF&W Project Solutions as the public personal brand
- Hermes as active

## Standard Boilerplate

Use this for client-facing materials:

```text
AXIS is developed by Wayne Francis and delivered through SF&W Project Solutions. It provides structured AI operating systems for SMEs, solo operators, and practical workflow automation.
```

## Compliance Note

Formal legal, privacy, contract, and invoice wording should use the correct legal/trading details before client use.
