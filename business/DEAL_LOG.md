## Format

| Date sourced | Deal | ICP match | Validation source | First action | Status | Outreach handoff | Pipeline ID |
|---|---|---|---|---|---|---|---|

## Status values

- `sourced` ‚Äî deal identified, validated, ready for outreach
- `outreach drafted` ‚Äî outreach agent has produced the message
- `contacted` ‚Äî message sent
- `replied` ‚Äî prospect replied
- `qualified` ‚Äî moved to pipeline
- `closed-won` / `closed-lost`
