# Relationship Memory

Status: active
Purpose: Consent-aware relationship memory for SF&W Project Solutions.

## Rule

Store only useful relationship context that Wayne has provided or approved.

Do not invent relationships.

Do not store sensitive personal data unless there is a clear business reason and Wayne confirms it should be stored.
