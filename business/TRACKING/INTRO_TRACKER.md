# Intro Tracker

Status: active
Purpose: Track warm introductions, referral requests, and relationship-led opportunities.

## Rule

No introduction should be treated as live until both sides have agreed.

Claude can draft intro requests, but Wayne decides whether to send them.

## Tracker

## Status Options

Use:
- idea
- ask Wayne
- ask party A
- ask party B
- both agreed
- intro sent
- declined
- no response
- complete
- archived

## Intro Request Template

```text
Hi [Name],

I thought of you because [specific reason].

[Person] is working on [relevant context]. I think there may be a useful conversation around [shared point].

No pressure either way - would you be open to an intro?

Wayne
```

## Completion Notes

After an intro is complete, record:
- whether it happened
- whether it created value
- whether follow-up is needed
- whether either person should stay in relationship memory
