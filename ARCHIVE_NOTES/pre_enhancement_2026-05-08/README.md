# Axis OS v3

A deployable AI operating system for SME clients. Three-layer architecture. Sole user interface is the Digital Chief of Staff (DCoS).

## Layers

1. **Command** — DCoS routes everything, tracks execution
2. **Thinking** — 14 Axis AI specialist agents do the work and challenge outputs
3. **Revenue** — Deal Sourcing OS finds deals, hands to outreach, tracked by pipeline

## Quick start

Type any task. DCoS routes it.

Special commands:
- `Status` — current state
- `Workstreams` — active work + next actions
- `Block list` — anything stuck
- `Next action` — single next concrete action
- `Begin Axis OS setup` — full client handover flow

## Folders

- `.claude/agents/` — 16 agents (14 specialists + dcos + deal-sourcing)
- `business/` — context, pipeline, clients, trackers, handover suite
- `docs/` — DCoS spec, Deal Sourcing OS spec, ecosystem map, OS onboarding guide
